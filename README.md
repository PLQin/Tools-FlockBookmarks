## ‚ú® Features

- The plugin is based on the Chrome browser. The search home page is referenced with Chrome.
- when the user opens a new empty TAB, it is the first page of the plug-in, then obtains the user authorization, assigns and saves the random ID to the browser extension, and exports the user's browser favorites to the server through the browser extension.
- multiple users join a common group by password (hash value) and share the favorites of all members in the group.
